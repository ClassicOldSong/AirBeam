# AirBeam

[English](README.md)

为您的 Android 设备打造的空气鼠标与触摸板。可用于 Miracast/有线投屏，为您的 Android 设备提供近似 DeX 的体验。配合 AR 眼镜效果更佳。独家支持锁屏状态下投屏显示（需要 Android 13+）。

本仓库仅用于提供支持。

[下载](https://github.com/ClassicOldSong/AirBeam/releases)
